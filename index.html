Madani Monitoring


Madani Monitoring is an Android application to monitor a project progress for CV. Madani Cipta Selaras. There are three types of the application, for admin, for subcontractor, and for supervisor. Admin can create a project and see the progress of each project. Admin can also preorder an item to a subcontractor. Then the subcontractor receive a notification about the preorder and they can see the detail order. Subcontrator can create a progress everytime they make a progress and supervisor can accept the proggress or reject it if there are some problem.

This is an app I created for a client while I was freelancing. When developed this app I looked around for similar apps for monitoring project and adapted it to meet the business process of the client’s company. I also conduct user testing to get some feedback from the client. When the app was ready, the client was very impressed because they always monitor a project manually and it requires a lot of papers and sometimes they lost the paper and this application helps them so much.

  • Date: April 2019 - August 2019
  • Client: CV. Madani Cipta Selaras
  • Skills: Android, Kotlin, Rest Api, MVVM

Nawarin


Nawarin is a marketplace made for two kind of platforms which are website and android application. The difference between Nawarin and another marketplace platform is Nawarin has features that benefits the reseller by enabling them to resell the product that the seller wants to sell easily. Nawarin application consists of two applications which are application that are used by buyers to get information about products being sold and also make online purchase transactions. The second application can be used by the seller to manage information on the goods he sells and also accept purchase orders from buyers.

This is an app that my team developed during our internship at Profile Image Studio. Profile Image Studio wants to create a new innovation that can make it easier for shop owners to sell their products while managing their resellers in an integrated system. My team’s task was to build the Android application for buyers. My friends designed the app user interface and I coded the app. My team also conduct a usability testing to ensure that users can understand and can easily operate the app. The app is still in an early stage of development so it still needs a lot of improvement.

  • Date: June 2019 - September 2019
  • Skills: Android, Kotlin, Rest Api, MVVM

Nowcan Apps


Nowcan Apps or Nowcan Indonesia is an online transportation service application in Jayapura Papua that is integrated with the Marketplace. This application allow users to order a ride, food, and send a package. In addition, Nowcan Apps also has a marketplace service that facillitates users to shop online from many merchants who have joined the service. This is the first online transportation service that operates in Papua. There is also Nowcan Driver that can be used by the driver to receive an order from users.

This is a project that I worked on while I was working as an Android developer at Innistudio. The application have been developed before by other team but it still had many bugs and the client wanted to change the business process so my task was to reengineering the application to meets the new requirement. I also refactoring the code because the previous code was too messy. I used Kotlin programming language and MVVM architecture to improve its maintanability.

  • Date: December 2018 - August 2019
  • Client: Nowcan Indonesia
  • Skills: Android, Kotlin, Rest Api, MVVM
  • Playstore

HaloMedis


HaloMedis is an Android application that provides health service features for users. HaloMedis is a liaison between healthcare providers and users by providing a directory of information on hospitals, doctors and clinics or specialist practices. With this application, users can book a clinic service or a hospital room also call an ambulance from the nearest hospital. This application was developed using Kotlin programming language and used Firebase for the database and to send push notification.

My Team developed this application for a competition that we participated in. At first, this application only has two features which are MediHospital for booking hospital room and AmbulanceNow for calling an ambulance from nearest hospital. We ended up didn’t win the competition but we tried to improve the app and then participated in another competition. We added a new feature that is MediClinic for booking a clinic service and build one more app for the clinic side. For the second competition we achieved a good result, we got a silver medal and one special award. Now we still trying to improve this app so that it can meet the needs of users in the health sector.

  • Date: September 2018 - April 2019
  • Skills: Android, Kotlin, Firebase
  • Article

SiagaDBD


SiagaDBD is an Android application to facilitate the survey process for dengue fever sufferers. Public health officers use this application to fill out forms according to environmental surveys or larvae surveys from a house or public facilities, then the admin can see the summary form that has been sent by the officer for further analysis and provide further action if needed. If further action is needed, the officer will receive a notification to conduct a health survey on dengue fever patients then fill out a form according to the result of the examination.

This is an app that my team developed for the Faculty of Medicine, University of Brawijaya. The client didn't really know what they wanted and they kept changing the requirement of the app so we asked them to extend the deadline due to major changes to the app. And then we conduct user testing to get some feedback from the client.

  • Date: April 2018 - June 2018
  • Client: Faculty of Medicine, Brawijaya University
  • Skills: Android, Java, Rest API, Firebase

Olimpiade Brawijaya Apps


Olimpiade Bwawijaya apps is an application that developed for the Olympic event held at Brawijaya University. This application can be used by the committee, athletes, supporters, and all UB students.
